# primes
Includes following tools to work with prime numbers:
* Sieve of Eratosthenes
* Sieve of Atkin 
* Lucas-Lehmer primility test for Mersenne numbers
* Miller–Rabin primality test
* Lucas primality test
* Baillie–PSW primality test
